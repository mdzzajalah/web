HTML - HyperText Markup Language

Struktur dasar setiap website

Apa itu HTML?

HTML adalah bahasa markup standar untuk membuat halaman web. HTML mendeskripsikan struktur halaman web dengan menggunakan elemen-elemen.

Fitur Utama

  • Struktur dokumen dengan tag
  • Hyperlink untuk navigasi
  • Form untuk input user
  • Semantic elements (header, nav, section, etc)
  • Embed multimedia

Kegunaan

  • Membangun struktur website
  • Membuat form input
  • Menampilkan konten multimedia
  • SEO Optimization
  • Web Accessibility

Contoh Kode HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Contoh HTML</title>
</head>
<body>
    <header>
        <h1>Selamat Datang!</h1>
        <nav>
            <a href="#home">Home</a>
            <a href="#about">About</a>
        </nav>
    </header>
    
    <main>
        <article>
            <h2>Artikel Pertama</h2>
            <p>Ini adalah contoh paragraf HTML</p>
            <img src="image.jpg" alt="Deskripsi gambar">
        </article>
    </main>
    
    <footer>
        <p>¬© 2024 Website Saya</p>
    </footer>
</body>
</html>

CSS - Cascading Style Sheets

Memberikan gaya dan layout pada website

Apa itu CSS?

CSS adalah bahasa stylesheet yang digunakan untuk mendeskripsikan presentasi dokumen HTML. CSS mengontrol layout, warna, font, dan animasi.

Fitur Utama

  • Selectors dan specificity
  • Box model (margin, padding, border)
  • Flexbox dan Grid layout
  • Animasi dan transisi
  • Responsive design (media queries)

Kegunaan

  • Styling halaman web
  • Layout responsif
  • Animasi dan efek visual
  • Theming dan branding
  • Print styling

Contoh Kode CSS

/* CSS Modern dengan Flexbox */

JavaScript

Bahasa pemrograman untuk web yang interaktif

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman tingkat tinggi yang digunakan untuk membuat halaman web interaktif. Dapat dijalankan di browser dan server (Node.js).

Fitur Utama

  • Event-driven programming
  • Asynchronous programming (Promises, async/await)
  • DOM manipulation
  • Closures dan hoisting
  • ES6+ features (arrow functions, classes, modules)

Kegunaan

  • Web interactivity
  • Single Page Applications (SPA)
  • Game development
  • Mobile apps (React Native)
  • Server-side development (Node.js)

Contoh Kode JavaScript Modern

// ES6+ Features
class User {
    constructor(name, email) {
        this.name = name;
        this.email = email;
    }
    
    greet() {
        return `Hello, ${this.name}!`;
    }
}

// Arrow functions & Array methods
const users = [
    new User('Alice', 'alice@email.com'),
    new User('Bob', 'bob@email.com')
];

const userNames = users.map(user => user.name);

// Async/Await example
async function fetchData() {
    try {
        const response = await fetch('https://api.example.com/data');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error:', error);
        return null;
    }
}

// DOM Manipulation
document.addEventListener('DOMContentLoaded', () => {
    const button = document.querySelector('#myButton');
    const counter = document.querySelector('#counter');
    let count = 0;
    
    button.addEventListener('click', () => {
        count++;
        counter.textContent = `Count: ${count}`;
        
        // Add animation
        counter.style.transform = 'scale(1.2)';
        setTimeout(() => {
            counter.style.transform = 'scale(1)';
        }, 200);
    });
});

// Modern array operations
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(n => n * 2);
const even = numbers.filter(n => n % 2 === 0);
const sum = numbers.reduce((acc, curr) => acc + curr, 0);

// Optional chaining & Nullish coalescing
const user = {
    profile: {
        name: 'John',
        address: {
            city: 'Jakarta'
        }
    }
};

const city = user.profile?.address?.city ?? 'Unknown City';

Python

Bahasa pemrograman serbaguna untuk berbagai keperluan

Apa itu Python?

Python adalah bahasa pemrograman tingkat tinggi yang mudah dipelajari dengan sintaks yang bersih dan mudah dibaca. Sangat populer untuk data science, AI, dan web development.

Fitur Utama

  • Syntax yang mudah dibaca
  • Dynamic typing
  • Comprehensive standard library
  • Object-oriented & functional programming
  • Banyak library pihak ketiga

Kegunaan

  • Web Development (Django, Flask)
  • Data Science & Machine Learning
  • Automation & Scripting
  • Game Development
  • Desktop Applications

Contoh Kode Python

# Python Modern dengan Type Hints dan f-strings
from typing import List, Optional
from dataclasses import dataclass
import asyncio

# Data class untuk OOP modern
@dataclass
class Product:
    name: str
    price: float
    category: str
    
    def display_info(self) -> str:
        return f"{self.name} - ${self.price:.2f} ({self.category})"

# List comprehension
products: List[Product] = [
    Product("Laptop", 999.99, "Electronics"),
    Product("Mouse", 25.50, "Accessories"),
    Product("Keyboard", 75.00, "Accessories")
]

# Filter dengan list comprehension
accessories = [p for p in products if p.category == "Accessories"]

# Generator expression
expensive_products = (p for p in products if p.price > 100)

# Async programming
async def fetch_product_data(product_id: int) -> Optional[dict]:
    """Fetch product data asynchronously"""
    await asyncio.sleep(1)  # Simulate API call
    return {"id": product_id, "name": f"Product {product_id}", "stock": 100}

# Context manager untuk file handling
def read_file_safely(filename: str) -> str:
    """Read file with automatic cleanup"""
    with open(filename, 'r', encoding='utf-8') as file:
        return file.read()

# Error handling
def calculate_discount(price: float, discount: float) -> float:
    """Calculate price after discount"""
    if discount < 0 or discount > 1:
        raise ValueError("Discount must be between 0 and 1")
    
    if price < 0:
        raise ValueError("Price cannot be negative")
    
    return price * (1 - discount)

# Main execution
async def main():
    print("=== PRODUCT CATALOG ===")
    
    for product in products:
        print(f"‚Ä¢ {product.display_info()}")
    
    # Async operations
    tasks = [fetch_product_data(i) for i in range(1, 4)]
    results = await asyncio.gather(*tasks)
    
    print("\n=== ASYNC RESULTS ===")
    for result in results:
        print(f"Fetched: {result}")

# Run async main
if __name__ == "__main__":
    asyncio.run(main())
